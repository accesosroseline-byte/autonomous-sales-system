# Skill: Responder Cliente

## Descripción

Genera una respuesta inicial para clientes nuevos o mensajes de saludo.

---

## Objetivo

- Iniciar conversación
- Generar interés
- Guiar al cliente hacia una intención de compra

---

## Pasos

1. Saludo cordial
2. Identificación de necesidad
3. Pregunta abierta para continuar conversación

---

## Ejemplo

"Hola 😊 ¡Bienvenida! Tenemos accesorios hermosos disponibles. ¿Qué tipo de producto estás buscando?"
