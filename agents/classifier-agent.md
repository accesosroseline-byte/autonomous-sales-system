# Classifier SubAgent

## Descripción

SubAgent encargado de analizar los mensajes del cliente y determinar su intención.

---

## Objetivo

Transformar texto libre en datos estructurados para facilitar la toma de decisiones del sistema.

---

## Output esperado

```json
{
  "intent": "compra | saludo | consulta | cierre",
  "category": "cat1 | cat2 | cat3 | cat4"
}
