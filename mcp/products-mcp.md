# MCP: Products

## Descripción

Módulo encargado de consultar productos desde la base de datos en Supabase.

---

## Función

- Obtener productos por categoría
- Filtrar productos activos
- Retornar información estructurada

---

## Datos disponibles

- Nombre
- Precio
- Categoría
- Stock

---

## Uso

Se utiliza principalmente en el skill de recomendación de productos.
