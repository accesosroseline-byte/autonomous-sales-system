# Autonomous Sales System

Sistema basado en agentes de inteligencia artificial para automatizar la atención y ventas en WhatsApp.

---

## 🎯 Problema

El proceso comercial actual es manual, reactivo y no aprovecha los datos disponibles, lo que limita la escalabilidad del negocio.

---

## 🧠 Solución

Se propone un sistema multi-agente que:

- Automatiza la atención al cliente
- Recomienda productos
- Mejora la conversión
- Sienta bases para inteligencia comercial

---

## ⚙️ Arquitectura

El sistema está compuesto por:

- Orchestrator (n8n)
- Sales Agent
- SubAgent de clasificación
- MCPs (datos desde Supabase)
